# QuickDownload

`通过 URL 抓取内容(HTML/API-JSON)快速下载自定义的资源`
Quickly download images from the remote

## Install

[Install NodeJS and suggest >= 8.11.0](https://nodejs.org/zh-cn/)

## Usage
```bash
npm start
```
